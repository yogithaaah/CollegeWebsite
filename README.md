# CollegeWebsite

This repository contains the source code for a simple college website built using **HTML** and **CSS**. The website showcases key information about Prasad V Potluri Institute of Technology (PVPSIT), including courses, testimonials, and contact details.

## Features
- **Responsive Design**: Ensures the website looks great on all devices.
- **Sticky Navigation**: Easy-to-access navigation menu with smooth scrolling.
- **Hero Section**: Welcoming banner with a call-to-action button.
- **About Section**: Highlights the institution's achievements and mission.
- **Courses Section**: Detailed cards for the courses offered, including images and descriptions.
- **Testimonials**: Displays feedback from alumni.
- **Contact Section**: Provides essential contact details for inquiries.

## Files
### `index.html`
- The main HTML file containing the structure of the website.
- Includes sections for Home, About, Courses, Testimonials, and Contact.

### `styles.css`
- The CSS file responsible for styling the website.
- Features:
  - Clean and modern layout with well-organized sections.
  - Custom styles for navigation, hero banner, course cards, and testimonials.
  - Responsive adjustments for better usability on different screen sizes.

## Preview
The website includes:
